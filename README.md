# 🎮 Unblocked Games Website

A simple, fun, and fully responsive unblocked games website built with HTML, CSS, and JavaScript.

✨ **Modern Design** - Clean and attractive UI with smooth animations
🎮 **12+ Games** - Browse a collection of popular unblocked games
🔍 **Search Functionality** - Easily find games by name or category
📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile
⚡ **Fast Loading** - Lightweight and optimized for quick load times
🎨 **Beautiful UI** - Gradient backgrounds and smooth hover effects

## Customization

### Add More Games
Edit `script.js` and add new games to the `games` array:

```javascript
{
    id: 13,
    name: 'Game Name',
    icon: '🎮',
    description: 'Game description',
    category: 'action'
}
```

### Change Colors
Edit `:root` CSS variables in `style.css`:

```css
:root {
    --primary-color: #6c5ce7;
    --secondary-color: #a29bfe;
    --accent-color: #00b894;
    /* ... more colors ... */
}
```
